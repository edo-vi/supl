Detailed Description

Namespace containing all the definitions and implementations of classes representing samples of labeled datapoints (Sample), samples with defined disjoint subsets (SlicedSample), samples with defined disjoint subsets with one that is the currently ´selected' subset (SlicedSettedSample), union of samples representing a training sample and a test sample (ValidationSample)

Parse the CSV file with the given path and generates a sample from it. The file must have the following structure: there must be no header and no row header, values must be separated by a comma and the label must be the last value of each row.
