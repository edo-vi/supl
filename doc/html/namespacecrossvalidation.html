Cross-validates the learner on the passed sample by testing all the parameters in the parameter set.

Parameters
T
learnerA Learner that will train and validate on the sample
paramsetA ParameterSet whose elements are used with the learner to calculate its error on the validation sample and from which the ‘best’ parameter is chosen
sampleThe sample used to cross-validate the learner
Returns
The ‘best’ parameter among those in the ParameterSet, that is, the one with the lowest average error
